1.
git commit -- amend: edit commit cuối cùng bằng cách mở editor
git commit -- amend -m "<message>": edit commit cuối cùng bằng cách sửa trực tiếp trong <message>
git restore -- staged <file>: chuyển file từ vùng staging area về vùng working directory
git reset HEAD~1: quay về commit trước commit cuối cùng
